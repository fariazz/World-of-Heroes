INSTALLATION NOTES - WORLD OF HEROES

PABLO C. FARIAS NAVARRO

1. PYTHON VERSION

WOH was developed and tested using Python 2.6.2 (release26-maint, Apr 19 2009, 01:58:18).

2. OPERATING SYSTEM

WOH was developed and tested under Ubuntu 9.04 Jaunty Jackalope. 

WOH was also tested in Windows XP Home Edition, with Python 2.5. Everything worked in this OS with the exception of the full screen mode.

3. REQUIRED PACKAGES

The following packages need to be installed to run WOH:

-sys: part of The Python Standard Library, installed by default.
http://docs.python.org/library/sys.html

-random: part of The Python Standard Library, installed by default.
http://docs.python.org/library/random.html

-pygame: can be downloaded from: 
http://www.pygame.org/download.shtml

installation instructions for pygame at: http://www.pygame.org/install.html

4. RUNNING WOH

WOH can be run from the command line or from IDLE, 

within the folder WOH type:

python game.py

WOH uses relative paths only, and universal newline support is enabled.
